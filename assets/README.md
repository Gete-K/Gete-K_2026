# 画像の置き場所

このフォルダには、公開サイトに載せてよい画像だけを置きます。

## 置いてよいもの

- 本人から掲載許可を取った劇団員写真
- Web用にリサイズ済みの画像
- EXIF、位置情報、撮影機種情報を削除した画像
- 公開されても問題ないキービジュアル、ロゴ、OGP画像

## 置いてはいけないもの

- 元画像、高解像度画像
- 未掲載・未許可の劇団員写真
- 稽古場や楽屋など、写り込み確認が終わっていない写真
- ファイル名に本名、住所、撮影場所などが含まれる画像

## おすすめ構成

```text
assets/
  public/
    branding/
      logo.png
    members/
      member-01.jpg
      member-02.jpg
    keyvisual.jpg
    ogp.jpg
```

`assets/private/`、`assets/originals/`、`assets/raw/` は `.gitignore` で除外しています。
ただし、除外されていても安心せず、公開して困る画像はリポジトリの外で管理してください。
